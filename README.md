# housing-data-price-prediction
Predict housing prices using machine learning. Analyze features like location, rooms, population, income, and proximity to the ocean. Includes EDA, preprocessing, and regression models to predict median house value using Python libraries like Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.
# Dataset
## Features:
longitude, latitude, housing median age, total rooms, total bedrooms, population, households, median income, ocean proximity
## Target: 
median house value
